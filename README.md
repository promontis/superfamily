# Superfamily

Familiegerichte leerplatform voor kinderen van 4-8 jaar.

## Over Superfamily

Superfamily combineert het beste van gaming, educatie en real-life ervaringen:

- **2.5D Platform Gaming** - Mario-stijl gameplay met Unreal Engine 5.5+
- **Educatieve Content** - Nederlandse curriculum (Taal, Rekenen)
- **Real-Life Missies** - Fysieke taken zoals "bestel bij de bakker"

## Tech Stack

| Component | Technologie |
|-----------|-------------|
| Game Engine | Unreal Engine 5.5+ |
| AI Development | Flopperam |
| Mobile App | React Native |
| Backend | Node.js + PostgreSQL |
| CMS | Strapi |

## Doelgroep

- Kinderen 4-8 jaar (Groep 1-4)
- Nederlandse gezinnen
- Ouders actief betrokken
