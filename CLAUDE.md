# Superfamily

Familiegerichte leerplatform voor kinderen van 4-8 jaar.

## Project Overzicht

Superfamily combineert:
- **2.5D Platform Gaming** - Mario-stijl gameplay
- **Educatieve Content** - Nederlandse curriculum (Taal, Rekenen)
- **Real-Life Missies** - Fysieke taken in de echte wereld

## Tech Stack

| Component | Technologie |
|-----------|-------------|
| Game Engine | Unreal Engine 5.5 |
| AI Development Tool | Flopperam (level building, blueprints) |
| Mobile App | React Native |
| Backend | Node.js + PostgreSQL |
| CMS | Strapi |

## Doelgroep

- Kinderen 4-8 jaar (Groep 1-4)
- Nederlandse gezinnen
- Ouders actief betrokken

## Kernconcepten

- **Wereldkaart**: Duolingo-stijl progressie met eilanden
- **Levels**: Side-scrolling 2.5D platformer met educatieve vragen
- **Real-Life Missies**: Taken zoals "bestel bij de bakker", foto-verificatie
- **Ouder Dashboard**: Voortgang volgen, missies goedkeuren
